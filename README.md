# 学生学术信息管理系统

# Система управления успеваемостью студентов

---

## 📖 项目简介 | Описание проекта

**中文**：学生学术信息管理系统是一个基于 **Spring Boot + React** 的 B/S 架构教务管理平台，支持学生、教师、管理员三种角色，实现成绩录入、审核、公示、归档的完整工作流，以及 GPA 计算、奖学金自动评定和学业排名功能。

**Русский**：Система управления успеваемостью студентов — это образовательная платформа на базе архитектуры **Spring Boot + React** (B/S), поддерживающая три роли: студент, преподаватель, администратор. Реализует полный рабочий процесс ввода, проверки, публикации и архивирования оценок, а также расчет GPA, автоматическое назначение стипендий и академический рейтинг.

## ✨ 核心功能 | Основные функции

### 🔐 用户认证与授权 | Аутентификация и авторизация
**中文**
- 基于 JWT 的无状态身份认证
- 三种角色权限控制：学生、教师、管理员
- 角色级别的路由保护

**Русский**
- Безусловная аутентификация на основе JWT
- Контроль доступа для трех ролей: студент, преподаватель, администратор
- Маршрутная защита на уровне ролей

### 📝 成绩管理 | Управление оценками
**中文**
- 教师录入学生成绩
- 支持补考标记与原始成绩记录
- 完整的审核工作流：录入中 → 已送审 → 已审核 → 已公示 → 已归档

**Русский**
- Преподаватель вводит оценки студентов
- Поддержка отметки пересдачи и записи исходных баллов
- Полный рабочий процесс проверки: ввод → на проверке → проверено → опубликовано → архивировано

### 🎓 奖学金评定 | Назначение стипендий
**中文**
- 基于 GPA 自动计算奖学金等级
- 一等奖学金（GPA ≥ 3.8）：¥5,000
- 二等奖学金（GPA ≥ 3.5）：¥3,000
- 三等奖学金（GPA ≥ 3.0）：¥1,000

**Русский**
- Автоматический расчет уровня стипендии на основе GPA
- Первая стипендия (GPA ≥ 3.8): ¥5,000
- Вторая стипендия (GPA ≥ 3.5): ¥3,000
- Третья стипендия (GPA ≥ 3.0): ¥1,000

### 📊 学业排名 | Академический рейтинг
**中文**
- 按班级或全年级进行 GPA 排名
- 评级展示：优秀 / 良好 / 中等 / 及格 / 不及格

**Русский**
- Рейтинг GPA по классу или по всему курсу
- Отображение рейтинга: отлично / хорошо / средне / удовлетворительно / неудовлетворительно

### 📄 电子成绩单 | Электронная ведомость
**中文**
- 学生可查看个人完整成绩单
- 仅展示已公示或已归档的成绩

**Русский**
- Студент может просмотреть полную электронную ведомость
- Отображаются только опубликованные или архивированные оценки

### 🌐 国际化 | Интернационализация
**中文**
- 支持中文 / 俄语双语切换

**Русский**
- Поддержка переключения между китайским и русским языками

| 表名 / Таблица | 说明 / Описание |
|:-:|:-:|
| `users` | 用户信息 / Информация о пользователях |

| `grades` | 成绩记录 / Записи оценок |
| `scholarships` | 奖学金记录 / Записи стипендий |
| `course_teachers` | 课程-教师关联 / Связь курсов и преподавателей |

### 表关系 | Связи таблиц

```
users 1──1 students
users 1──1 teachers
students 1──N grades N──1 courses
students 1──N scholarships
courses 1──N course_teachers N──1 teachers
```

## 🛠️ 快速开始 | Быстрый старт

### 前置条件 | Предварительные требования
- **JDK 21+**
- **Node.js 18+**
- **PostgreSQL 16+**
- **Maven 3.9+**

### 1. 数据库初始化 | Инициализация базы данных

```sql
-- 使用 psql 或 DBeaver 执行
CREATE DATABASE yxw;
-- 然后运行 init.sql 创建表结构
```

### 2. 启动后端 | Запуск бэкенда

```bash
cd yxwbackend
mvn spring-boot:run
```

后端启动后，`DataInitializer` 会自动创建测试用户、课程和成绩数据。

После запуска бэкенда `DataInitializer` автоматически создаст тестовых пользователей, курсы и данные оценок.

### 3. 启动前端 | Запуск фронтенда

```bash
cd yxwfrontend
npm install
npm start
```

前端运行在 `http://localhost:3000`

Фронтенд запускается на `http://localhost:3000`
